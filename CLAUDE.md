# CLAUDE.md

This file provides guidance to Kilo Code when reading and writing code.

## Project Overview
This is a React pages demo project showcasing various React components and patterns.

## Tech Stack
- React with TypeScript/JavaScript
- Vite as build tool
- ESLint for code quality

## Code Review Guidelines
- Review TypeScript files for type safety
- Check React component patterns follow best practices
- Ensure proper imports and exports
- Verify accessibility in components

## Commands
- `npm install` - Install dependencies
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
